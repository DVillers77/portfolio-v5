Systems-Driven Engineering

This gallery documents my transition from auditing physical systems to engineering digital ones. Every project here has been individually audited for structural integrity, semantic clarity, and technical scalability. I treat code as a system that must be as resilient and intuitive as the human body.

From implementing recursive AI algorithms to managing complex asynchronous data cycles, my focus remains on clean code architecture (BEM/DRY) and strict adherence to WCAG AA accessibility standards. Explore the collection below to see how I translate technical rigor into seamless, human-centered digital solutions.

Modern Tic-Tac-Toe: Advanced Logic & State Management

I engineered this "Cyber-Theme" application to demonstrate recursive logic and defensive programming in Vanilla JavaScript. The centerpiece is an unbeatable "Hard Mode" powered by the Minimax algorithm, which evaluates all possible game outcomes to ensure optimal computer moves.

To enhance the user experience, I integrated the HTML5 Canvas API for dynamic win-line animations and implemented LocalStorage for persistent session state, ensuring user scores remain intact across page refreshes. This project represents the intersection of robust back-end logic and high-contrast, performant UI design.

API Fun Center: Asynchronous Data Integration

I developed this application to master Asynchronous JavaScript (AJAX) and the Fetch API. The project demonstrates the ability to integrate multiple third-party REST APIs, handling complex data types ranging from JSON text strings to dynamic image assets. I focused heavily on error handling and input sanitization (using encodeURIComponent) to ensure the application remains stable even when external services return unexpected results.

To enhance the user experience, I implemented dynamic UI feedback, including loading states and fallback messaging for screen readers. By managing the asynchronous lifecycle of four distinct data sources—including a translator and random fact generators—this project showcases my ability to bridge the gap between back-end data services and performant, responsive front-end interfaces.
